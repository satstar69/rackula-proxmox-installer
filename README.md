# rackula-proxmox-installer
Automated LXC installer for Rackula on Proxmox VE

Run the following command in proxmox shell to install Rackula:

bash -c "$(wget -qO- https://raw.githubusercontent.com/satstar69/rackula-proxmox-installer/main/rackula-lxc.sh | tr -d '\r')"
